# $guildFetch
Fetch a guild property.
## Usage
> `$guildFetch[property;guild id?]`
## Parameters
|   Name   |             Description              |  Type  |  Default value   |
|----------|--------------------------------------|--------|------------------|
| Property | Guild property name.                 | String | none             |
| Guild ID | Guild ID to fetch the property from. | String | d.ctx?.guild?.id |

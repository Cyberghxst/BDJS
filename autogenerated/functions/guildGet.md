# $guildGet
Get a guild property.
## Usage
> `$guildGet[property;guild id?]`
## Parameters
|   Name   |            Description             |  Type  |  Default value   |
|----------|------------------------------------|--------|------------------|
| Property | Guild property name.               | String | none             |
| Guild ID | Guild ID to get the property from. | String | d.ctx?.guild?.id |

# $userFetch
Fetch an user property.
## Usage
> `$userFetch[property;user id]`
## Parameters
|   Name   |             Description             |  Type  |   Default value   |
|----------|-------------------------------------|--------|-------------------|
| Property | User property name.                 | String | none              |
| User ID  | User ID to fetch the property from. | String | d.ctx?.author?.id |
